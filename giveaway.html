Welcome to the Giveaway Tool

Your customizable application for running EVE Online giveaways on Twitch.

Download RustyBot

Features

Real-time Chat Integration

Connects directly to your Twitch channel for live entry collection based on customizable commands.

Dynamic Winner Animations

Engage your viewers with high-fidelity, customizable animations for winner selection.

ESI Pilot Verification

Automatic lookups and ESI verification to ensure the winner is a real EVE Online pilot.

Streamer Commands

Command Description
!draw Viewer command to enter the active giveaway (customizable).
!ign <name> Allows the winner to manually provide their In-Game Name.
/prize <item> Streamer command to set the current prize and donator.
/open Streamer command to open the giveaway for entries.
/close Streamer command to close entries and draw a winner.

RustyBot Giveaway is an unofficial EVE Online tool. Not affiliated with CCP Games.
